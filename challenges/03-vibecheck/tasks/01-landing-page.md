## Coding prompt
Write the HTML for the hero section of a landing page for a daily journaling app called "Margins." Include a heading, a one-line value proposition, and a way for the user to start.

## Taste profile

### register
A page that holds the door open — invites without asking for anything in return.

**Keep:**
- lowercase headers and copy that read as quiet statements, not slogans
- a single sentence stating the value, no more
- the call-to-action as a text link, not a button shape
- generous negative space — the page admits it has nothing to sell
- warm neutrals (stone, sand, paper) over cool greys

**Avoid:**
- all-caps brand names or shouting headlines
- pill-shaped CTAs with conversion-optimized copy ("Start Free Trial!")
- gradient backgrounds and decorative blur shapes
- testimonial logos or trust badges in the hero
- exclamation marks anywhere

### typography
A serif body for prose — "this is meant to be read, not scanned."

**Keep:**
- serif font family (Georgia, EB Garamond, source serif)
- light weight, generous line height
- the heading is one to two stops larger than the body, not five

**Avoid:**
- bold sans-serif display headings
- 5xl+ heading sizes
- letter-spacing tricks (uppercase tracking-widest)

## Judging rubric
The judge should reward outputs where:
1. The hero is one paragraph or one sentence, not a column of stacked components.
2. The CTA is a text link with inviting copy, not a styled button with conversion language.
3. The font choice is serif and the heading is restrained.
4. The whole composition feels like an invitation, not a sales page.

The judge should penalize outputs where:
1. There are multiple buttons in the hero, or any all-caps text.
2. The heading is "GET STARTED" / "JOIN NOW" / "TRANSFORM YOUR..." energy.
3. There are decorative gradients, hero images, or exclamation marks.
4. The composition reads as a SaaS landing page.

Cite specific lines or attributes from the output. "It uses serif" is too vague; "uses Georgia for the body and `text-2xl font-light` for the heading" is the kind of citation we want.
